# Image-Hosting-Website
A website created for image uploading, storage and managing. Powered by a React frontend, a nodejs express server, and AWS S3 buckets.

Tools:  
* React.js
* Node.js
* AWS S3, EC2, LoadBalancer, Route 53
* MySQL Database
